# 🚀 クイックスタート — Vercelアップロード（3ステップ）

## ⚡ 最速セットアップ手順

### ステップ1️⃣: GitHubリポジトリを作成（1分）

1. https://github.com にログイン
2. **右上の「+」** → **「New repository」**
3. **Repository name** に `glukkon-survey-report` と入力
4. **Create repository** をクリック

### ステップ2️⃣: ファイルをアップロード（1分）

1. 作成したリポジトリを開く
2. **Add file** → **Upload files**
3. 以下の4ファイルをドラッグ&ドロップ：
   - `index.html`
   - `vercel.json`
   - `package.json`
   - `README.md`
4. **Commit changes** をクリック

### ステップ3️⃣: Vercelでデプロイ（1分）

1. https://vercel.com にアクセス
2. **GitHubでサインアップ/ログイン**
3. **New Project** をクリック
4. GitHubから `glukkon-survey-report` を選択
5. **Deploy** をクリック

---

## ✅ 完了！

数秒後に以下のようなURLが発行されます：

```
https://glukkon-survey-report.vercel.app
```

このURLを開くと、アンケート分析レポートが表示されます！

## 📝 データを更新したい場合

1. **index.html** をテキストエディタで開く
2. データの部分を編集（グラフの数値など）
3. GitHubにプッシュ
4. Vercelが自動デプロイ → 完了！
