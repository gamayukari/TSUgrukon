# グルコン講座 アンケート分析レポート

講座生の満足度・理解度・成長を可視化するインタラクティブなアンケート分析レポート。

## 📊 レポートの内容

### 1. 全体分析
- **回答者数**: 40名
- **平均満足度**: 4.4/5.0
- **高満足度率**: 85%（4点以上）
- **トップスコア**: 62.5%（5点評価）

### 2. 満足度分析
- スコア別の分布を可視化
- グラフに基づく分析

### 3. 学習テーマ分析
- コンセプト設計・ポジショニング（18件）
- 行動・実践（18件）
- コミュニティの力（16件）
- 原体験・個性の発掘（15件）
- 発信・言語化（14件）
- マインドセット（12件）
- 商品・販売方法（10件）

### 4. 講座生の声
- 特筆すべき個人ストーリーを詳細に紹介
- ブレークスルー、成長、気づきの3カテゴリー

### 5. まとめ
- 主な成果の整理
- 改善提案
- 今後のステップ

## 🚀 Vercelでの利用方法

### 1. リポジトリをクローン
```bash
git clone https://github.com/yourusername/glukkon-survey-report.git
cd glukkon-survey-report
```

### 2. Vercelにデプロイ
```bash
# Vercel CLIをインストール（まだの場合）
npm i -g vercel

# デプロイ
vercel
```

または、Vercelダッシュボードで直接リポジトリを接続できます。

### 3. 自動デプロイの有効化
GitHubにプッシュすると、Vercelが自動的にデプロイします。

## 📝 カスタマイズ

### データの更新方法

`index.html`内の以下の部分を編集：

#### チャート1（満足度分布）
```javascript
data: {
  labels: ['5点', '4点', '3点', '2点'],
  datasets: [{
    data: [25, 9, 4, 2],  // ← ここを編集
  }]
}
```

#### チャート2（テーマ分析）
```javascript
data: {
  labels: ['コンセプト設計...', /* ... */],
  datasets: [{
    data: [18, 18, 16, 15, 14, 12, 10],  // ← ここを編集
  }]
}
```

### 色の変更

`:root`セクションのCSS変数を編集：
```css
:root {
  --primary-color: #185FA5;  /* メインカラー */
  --success-color: #1D9E75;  /* サクセス色 */
  /* その他のカラー... */
}
```
